# SP. Hardware Requirements

## Sim Rig (Digital Twin Workstation)
- GPU: NVIDIA RTX 4070 Ti or higher
- CPU: Intel i7 (13th Gen+) or AMD Ryzen 9
- RAM: 64 GB DDR5
- OS: Ubuntu 22.04 LTS

## Edge Kit (Physical AI)
- NVIDIA Jetson Orin Nano (8GB) or Orin NX (16GB)
- Intel RealSense D435i/455 camera
- USB IMU (BNO055)
- USB microphone/speaker array (ReSpeaker)

- Proxy: Unitree Go2 Edu (~$1,800)
- Miniature Humanoid: Hiwonder TonyPi Pro (~$600)
- Premium: Unitree G1 Humanoid (~$3,000)
